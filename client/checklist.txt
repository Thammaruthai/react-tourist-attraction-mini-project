Must Have
pass 1.พิมพ์คำค้นหาลงในช่อง Input เพื่อค้นหาสถานที่ท่องเที่ยว
pass 2.เปิด Website ขึ้นมาแล้ว User ยังไม่ได้พิมพ์ข้อความลงใน Input ให้เอาข้อมูลจาก Server ทั้งหมดมาแสดงได้เลย
pass 3.ผลลัพธ์รายการสถานที่ท่องเที่ยวที่แสดงบน UI จะต้องไปขอข้อมูลจาก Server
pass 4.รายละเอียด Description จะต้องแสดงไม่เกิน 100 ตัวอักษร
pass 5.เมื่อ User คลิกปุ่ม “อ่านต่อ” จะต้องเปิดลิ้งค์ขึ้นมาใน Tab ใหม่ของ Web Browser ลิ้งค์จะเอามาจาก Property url
